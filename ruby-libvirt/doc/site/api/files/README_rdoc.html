Ruby bindings for libvirt

The module Libvirt provides bindings to libvirt

The various +*Ptr+ types in Libvirt map loosely to the following Ruby classes:

virConnectPtr
Libvirt::Connect
virDomainPtr
Libvirt::Domain
virDomainInfoPtr
Libvirt::Domain::Info
virNetworkPtr
Libvirt::Network
